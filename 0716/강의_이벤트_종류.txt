이벤트 종류
	console.dir(window)
		-> on으로 시작하는 속성 -> 이벤트 명
		
	클릭
		click, onclick
		dbclick, ondbclick
		touch(모바일)
		
		마우스 관련 이벤트
		mouseenter - 마우스 커서가 요소에 들어갔을 때 (본인 요소)
		mouseleave - 마우스 커서가 요소에서 나갔을 때 (본인 요소)
		
		mouseover - 마우스 커서가 요소에 들어갔을 때 (본인 + 자식 요소)
		mouseout - 마우스 커서가 요소에서 나갔을 때 (본인 + 자식 요소)
		
		mousedown - 마우스 버튼을 눌렀을 때
		mouseup - 마우스 버튼을 눌렀다, 뗄 때
		
		mousemove - 마우스 커서를 움직일 때
		
		mousewheel - 마우스 휠을 조작할 때
	
	
	
	키 이벤트
		keydown - 키보드 키를 눌렀을 때 (한번만 실행) 
						-> 거의 모든 키
		keypress - 키보드 키를 눌렀을 때 (키를 누르고있는 동안 계속 실행)
						-> Ctrl, Alt, Shift는 동작하지 않는다
		
		keyup -  키보드 키를 눌렀다 뗐을 때
		
	
	기타
		scroll - 창을 스크롤 했을 때 발생
		resize - 창의 사이즈를 줄였을 때
		
		focus - 입력 항목에 커서가 가있을 때
		blur - 입력 항목에서 커서가 제거되었을 때
		
		change - select 태그에서 선택을 했을 때, file을 선택했을 때
		
		
	추가적인 이벤트 확인
	console.dir(window)로 확일 했을 때, on으로 시작하는 속성 -> 이벤트
